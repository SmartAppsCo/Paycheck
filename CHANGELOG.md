# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-01-18

### Added

- Initial open source release
- Offline-first licensing with signed JWTs
- Email-based activation (no permanent license keys)
- Multi-tenant organization structure
- Stripe and LemonSqueezy payment integrations
- Envelope encryption for sensitive data
- Operator and organization APIs
- Rust and TypeScript SDKs
- Rate limiting and audit logging
